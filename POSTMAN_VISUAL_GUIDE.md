## جدول Status Codes

| الكود | المعنى | مثال |
|------|-------|------|
| 200 | OK ✓ | GET request نجح |
| 201 | Created ✓ | POST request أنشأ سجل جديد |
| 204 | No Content ✓ | DELETE request نجح |
| 400 | Bad Request ✗ | بيانات خاطئة في Body |
| 401 | Unauthorized ✗ | بدون توثيق أو بيانات خاطئة |
| 403 | Forbidden ✗ | صلاحيات غير كافية |
| 404 | Not Found ✗ | السجل غير موجود |
| 500 | Server Error ✗ | خطأ في السيرفر |

## خطوات سريعة

### 1️⃣ للـ GET (عرض)
```
1. اختر GET من القائمة
2. أدخل الـ URL
3. اختر Authorization: No Auth
4. اضغط Send
```

### 2️⃣ للـ POST (إنشاء)
```
1. اختر POST من القائمة
2. أدخل الـ URL
3. اختر Authorization: Basic Auth
4. أدخل Username و Password
5. اذهب إلى Body > raw > JSON
6. أدخل البيانات
7. اضغط Send
```

### 3️⃣ للـ PUT (تحديث)
```
1. اختر PUT من القائمة
2. أدخل الـ URL مع ID (مثل /1)
3. اختر Authorization: Basic Auth
4. أدخل Username و Password
5. اذهب إلى Body > raw > JSON
6. أدخل البيانات المحدّثة
7. اضغط Send
```

### 4️⃣ للـ DELETE (حذف)
```
1. اختر DELETE من القائمة
2. أدخل الـ URL مع ID (مثل /1)
3. اختر Authorization: Basic Auth
4. أدخل Username و Password
5. اضغط Send (بدون Body)
```
